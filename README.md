# Pokedex Speedrun

Voice-first Gen 1 Pokemon speedrun game (151 every run) built with Next.js + TypeScript.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Required local assets

Place PNG files here:

`/Users/jonathangarnett/Documents/New project/pokedex-speedrun/public/assets/gen1/`

Expected filenames:

- `001.png`
- `002.png`
- ...
- `151.png`

The app does not fetch images from the internet.

## Build

```bash
npm run build
```
